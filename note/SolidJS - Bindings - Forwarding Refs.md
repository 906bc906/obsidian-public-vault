Ref 전달

컴포넌트 내부의 ref를 부모에게 노출하고 싶은 경우가 많이 있는데, props.ref 를 활용하여 ref를 부모에게 전달할 수 있다.

```tsx
//부모 컴포넌트
...
	return <Canvas ref={canvas} />;

//자식 컴포넌트 (Canvas)
export default function Canvas(props) {
  return <canvas ref={props.ref} width="256" height="256" />;
}
```

<!--ankiA-->

```tsx
//부모 컴포넌트
...
	return <Canvas ref={canvas} />;

//자식 컴포넌트 (Canvas)
export default function Canvas(props) {
  return <canvas ref={props.ref} width="256" height="256" />;
}
```
